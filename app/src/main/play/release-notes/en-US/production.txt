This release improves empty-state handling for paginated lists and fixes tag sorting, shared-folder icons, and default labels. Autofill security is strengthened by enforcing that a web domain is present for filled fields and consistent across the entire view tree. The update also includes a routine dependency bump and further MVI refactoring (details screens for groups, tags, locations, and folders) and a fix for data refresh not stopped on time on some devices with aggressive batter saver.
