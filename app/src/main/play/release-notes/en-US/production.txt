# Passbolt 1.21.0-31 Android app release notes

## Added
- Password generator policies: generate passwords according to set policies (character sets, length, etc.)
- Passphrase generation support if policy set
- Support emoji in passwords
- Updated entropy calc
- Added option to exclude look alike characters
- Added external service check for password breaches
- Expiry policies

## Fixed
- Fixed Play Store crash when data refresh finishes when no UI context available
